## Headings
```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

## Text Formatting
```markdown
*italic* or _italic_
**bold**
***bold and italic***
~~strikethrough~~
```

## Lists

**Unordered list**
```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

**Ordered list**
```markdown
1. First
2. Second
3. Third
```

## Links & Images
```markdown
[OpenAI](https://openai.com)

![Alt text](https://example.com/image.png)
```

## Blockquotes
```markdown
> This is a blockquote
```
